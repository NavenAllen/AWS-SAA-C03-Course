## Important Values
### Default VPC
- CIDR range always - 172.31.0.0/16
- In each subnet - /20
- Everything assigned public IPv4 by default

### S3
- Maximum Size: 5TB
- Unlimited Objects
- Flat structure
- Names are globally unique
- 100 buckets soft limit, 1000 hard limit per account

### CloudFormation
- Description MUST follow TemplateFormatVersion
